## ✅ How to Start (Install Python + IDE)

To run the codes in this repository, first install Python and any one IDE/editor.

### 1) Download Python (Required)
Download from official website:
- https://www.python.org/downloads/

✅ While installing: **tick "Add Python to PATH"**

---

### 2) Download any one IDE / Editor (Choose 1)

You can use any one of these:

#### ✅ Option A: PyCharm (Recommended for Python)
- https://www.jetbrains.com/pycharm/download/?section=windows

#### ✅ Option B: Visual Studio Code (Most Popular)
- https://code.visualstudio.com/download

---

## ▶️ How to Run a Python File

### Method 1: Using VS Code / PyCharm
- Open the project folder
- Open the `.py` file
- Click **Run**

### Method 2: Using Terminal / CMD
1. Open terminal in the folder
2. Run:

python filename.py
